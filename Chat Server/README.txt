
Example of Chat Server based on Socket.io

INSTRUCTIONS: 
  Register as a user before sending any messages
  
  Can either send global messages to all registered users,
    or private messages ('recipient_name : message').

  Can also send group messages, ('recipient_name1, recipient_name2 : message')

To run first install the dependencies in package.json by executing:
  >npm install

Then run:
  >node server.js

To try the application open several browser instances at the address:
  http://localhost:3000/chatClient.html
